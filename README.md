## Download & Installation
To install via Git:

    cd "~/Library/Application Support/TextMate/Bundles"
    git clone git://github.com/vigo/textmate-twitterbootstrap.tmbundle.git
    osascript -e 'tell app "TextMate" to reload bundles'
